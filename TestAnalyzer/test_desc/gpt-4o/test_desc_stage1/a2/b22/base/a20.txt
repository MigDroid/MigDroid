Functionality Under Test:
The script tests adding and deleting a note in the ToDo List app.
Interaction Steps:
1. Interaction: Click Allow button to grant notification permission.
2. Interaction: Click Add button to create a new note.
3. Interaction: Click text field to focus on note input.
4. Interaction: Input 'Sample Todo' text field to add note content.
5. Interaction: Click Navigate up button to return to main screen.
6. Interaction: Long press 'Sample Todo' note to select it.
7. Interaction: Click Delete button to remove the selected note.
8. Interaction: Click OK button to confirm note deletion.
Stop Condition:
Got the observation that a text subject with 'Sample Todo' has Disappear, it is a note, which means the note was successfully deleted.
