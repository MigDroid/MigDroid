Functionality Under Test:
Adding a new task to the task list in the ToDo List app.
Interaction Steps:
1. Interaction: Click Allow button to grant notification permission.
2. Interaction: Click Skip button to bypass initial setup.
3. Interaction: Click Tasks list to open task list.
4. Interaction: Click Add button to create a new task.
5. Interaction: Input Sample Todo text field to enter task name.
6. Interaction: Click Add button to save the task.
Stop Condition:
Got the observation that a TextView subject with text "Sample Todo" has Appeared, it is a task item, which means the task was successfully added.
